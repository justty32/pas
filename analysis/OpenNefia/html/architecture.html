三大支柱協作關係

ECS 是骨架

所有遊戲物件 = Entity + Components；所有行為 = EntitySystems 訂閱事件處理。

IoC 是黏合劑

EntityManager、MapManager、PrototypeManager 等皆以介面註冊，系統間靠 [Dependency] 取得彼此。

原型是燃料

YAML 原型在啟動時載入為記憶體資料，SpawnEntity(protoId, coords) 據此組裝出帶 Components 的實體。
