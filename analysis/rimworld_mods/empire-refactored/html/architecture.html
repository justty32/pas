核心架構與資料模型

單一真相 FactionFC（WorldComponent）持有整個帝國；每個聚落是正規 RimWorld WorldObject WorldSettlementFC。本頁綜整 00_overview.md 與 01_core_systems.md。

一句話定位

「世界地圖層的帝國經營」mod：玩家把單一殖民地擴張成多個自治附庸聚落（每聚落＝世界地圖 WorldSettlementFC），聚落自動產資源、定期向玩家繳稅，玩家可升級聚落／蓋據點建築／頒布派系敕令／派遣傭兵打 NPC 或防守，並透過事件做抉擇。

重構重點：把「聚落類型／資源／建築／敕令／事件」全面 def 化（XML 可擴充），並提供一整套 Registry + Interface 擴充接點，讓 submod 多數情況「不必 Harmony patch」。

入口 FactionColoniesMod : Mod（FactionColonies.cs:811）；Harmony 啟動 HarmonyPatcher（HarmonyPatches/HarmonyPatcher.cs:11）。

資料模型：FactionFC（單一真相）↔ WorldSettlementFC（每聚落）

FactionFC : WorldComponent, ILifecycleParticipant

唯一 WorldComponent，隨存檔序列化（ExposeData :278），持整個帝國狀態。

  • settlements: List<WorldSettlementFC>（:81，LookMode.Reference 序列化）
  • policies / factionTraits（:156-157）敕令與派系特性
  • eventManager（:183-188）事件清單
  • Bills / OldBills（:192-193）待結算稅單
  • resourcePools / factionResources（:197-200）派系級資源池
  • capitalLocation / TaxMap（:32-75）首都與稅金落地地圖

FactionFC.cs:12（2485 行）

WorldSettlementFC : Settlement

每聚落本身是世界地圖 WorldObject，獨立 ExposeData（:481）。

  • 四維狀態 unrest / loyalty / happiness / prosperity（:78-94）
  • settlementLevel（:41）＋升級狀態（:125-127）
  • settlementDef => def as WorldSettlementDef（:280）決定「類型」
  • 收益快取 totalIncome/Upkeep/Profit（:132-140，lazy）
  • 軍事 comp WorldObjectComp_SettlementMilitary

Worldobjects/WorldSettlementFC.cs:19（1923 行）

模型要點：帝國「多據點」不是塞進一個自訂清單欄位，而是每個據點都是正規 RimWorld WorldObject（可在世界地圖看到、有 Map），FactionFC.settlements 只是 by-reference 索引。

各子系統一覽

聚落建立

ColonyUtil.CreateSettlement（Util/ColonyUtil.cs:30）：MakeWorldObject → SetFaction → AddSettlement → SettlementTypeExtension.PostCreation → LifecycleRegistry.InvokeOnSettlementCreated。費用＝base + 每既有聚落 500。

資源系統

類型由 ResourceTypeDef : Def（純資料）定義：isDefaultResource / canTithe / isPoolResource、tech/research 限制、產量綁 FCStatDef。進階走 ResourceProduction/FilterExtension。

敕令（Edicts）

Social / Tax / Military 三類，隨等級解鎖。def＝FCPolicyDef（純資料）；行為＝FCPolicyBehavior + 13 子類，經 DefModExtension 掛載，AddTax 回呼 OnTaxCollected。

事件系統

def＝FCEventDef（觸發權重/時間、四維門檻、resource/research 需求）；處理 FCEventMaker.ProcessEvents（每 250 tick）+ MakeRandomEvent（每日）。選項/效果走 FCEventHandlerExtension。

派遣 / 軍事

每聚落掛 WorldObjectComp_SettlementMilitary（派遣 squad、CaravanDefend 防守）。Military/（23 檔）：MilitaryForce/MercenarySquadFC/SimulateBattleFC/ThreatScalingUtil。排程 MilitaryTick。

升級 / 建築 / 道路

升級 UpgradeSettlement（:696），上限雙重夾限。建築 BuildingFCDef（升級樹/前置/型別白名單）。道路用 MST 取代蛛網，roadBuilder.RoadTick() 驅動。
