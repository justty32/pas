opennefia-cpp = 把 OpenNefia C# 引擎核心的「反射依賴注入 / 系統掃描自動發現 / DataField 序列化」三大 C# 專屬特性，以 C++20 的 entt::type_list + fold expression、自由函式明確 vector 登錄、SystemCtx 顯式傳遞取代，在 medps 已驗證架構上重現同樣意圖，不依賴任何前端。
C++ 沒有反射，三大問題的實際解法

① 依賴注入

C# [Dependency] → SystemCtx 顯式傳遞（系統為無狀態自由函式，根本不需要 DI）；全域服務用 ServiceContext 強型別單例。

② 序列化

C# [DataField] 反射 → AllComponents type_list + fold expression + cereal adapter。新增 component 型別只改一行清單，存讀自動覆蓋。

③ 系統發現

C# Assembly 掃描 → add_system(lambda) 顯式 vector 登錄。順序即執行序，無啟動期反射開銷，可斷點可預測。
