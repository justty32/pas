AnimationsWithSounds

  • 骨骼動畫與音效軌打包為一體
  • 幀精確控制：音效與動畫幀同步觸發
  • 避免音畫脫節，提升打擊感一致性
空白鍵語境感知

跳躍 vs 閃避分流

  • 跑步中按空白鍵 → 跳躍
  • 靜止 / 走路時按空白鍵 → 閃避
  • 分流條件：movement/conditions/running 狀態

競態條件：同幀鬆開方向鍵時按空白鍵 → running 已被清除 → 觸發閃避而非跳躍

右攻擊動畫缺失
Bug：player.gd:102 呼叫 attack("right_attack_0")， 但該參數被忽略。AnimationTree 中只有 left_attack_0 節點， 右攻擊輸入靜默地播放左攻擊動畫。

修正方向：移除右攻擊輸入按鍵綁定，或補足完整的右攻擊 AnimationTree 節點。
連按攻擊鎖定

無自環（self-loop）機制

  • 狀態機未設定攻擊自環：攻擊期間無法重新進入攻擊狀態
  • 攻擊期間設 attacking = true 的邏輯被忽略，flag 不起作用
  • 實際效果：攻擊動畫播完後才接受下一次輸入，無法連段
怪物美術

龍模型碰撞體

  • 33 個 ConvexPolygonShape3D 組成複合碰撞體
  • AnimationTree 雙層狀態機：移動層 / 攻擊層獨立
  • 感嘆號動畫：combat=true 立即設定，怪物同步開始追蹤玩家

怪物死亡流程

  • monster.gd:76：call_deferred("set_script", preload("res://src/interact/monster drop.gd"))
  • 碰撞體保留（仍 enabled，死後仍可被偵測）
  • monster drop.gd 無 _ready()，透過 gathering.gd 提供掉落邏輯
  • queue_free() 清除整個節點樹
火焰攻擊時序
粒子生命週期：
t = 0.0s：emitting = true（開始噴火）
t = 3.5s：emitting = false（停止新粒子）
舊粒子仍存活至 lifetime = 5s（視覺拖尾效果）

RayCast 冷卻：1000ms — 玩家快速進出火焰區域時不會重複受傷
