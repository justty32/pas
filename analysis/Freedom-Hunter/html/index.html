五大設計亮點

動畫驅動遊戲邏輯

AnimationTree 不只播動畫，同時驅動移動速度。move_entity() 讀取當前動畫狀態節點決定速度倍率（walk=5, run=7.5, dodge=8），動畫狀態即是遊戲狀態。

雙層 AnimationPlayer 音畫同步

AnimationsWithSounds 把骨骼動畫 track 與 audio track 打包在同一個 AnimationPlayer 中，音效時機精確到幀（如喝藥水 t=0.35s 才響），可純用編輯器調整，不需改程式碼。

動態腳本替換

怪物死後不從場景移除，而是呼叫 set_script() 直接替換為 monster drop.gd，使節點搖身成為採集互動物件，既省效能又保留屍體位置。

MultiMesh 草地（1000 株一次 Draw Call）

Planter + GrassFactory + Shader 三層分工。Planter 以 @tool 在編輯器即時預覽；每株草的寬高、搖擺幅度存入 INSTANCE_CUSTOM.rgba 傳入 Shader；Worley 噪聲模擬陣風，強風時 ROUGHNESS 下降產生閃光感。

空白鍵語境感知操作

跑步中按空白 = 跳躍；靜止/走路時按空白 = 閃避，完全符合 Monster Hunter 原作設計。靠 movement/conditions/running 狀態分流，一鍵兩用。
