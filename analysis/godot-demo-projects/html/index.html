godot-demo-projects — Godot 官方範例集合

Godot 官方維護的範例專案集合，非單一應用程式，而是「分類編目 + 各自獨立可執行」的教學素材庫。每個含 project.godot 的資料夾都是一個獨立 demo，可用 Godot 引擎個別開啟或一次 Scan 全部匯入。

如何執行單一 demo
方式 A

個別開啟

啟動 Godot 4.6（C# demo 需 .NET 版）→ Project Manager 點 Import → 選某 demo 內的 project.godot（例 2d/platformer/project.godot）→ 按 F5 執行主場景。

方式 B

一次匯入全部

Project Manager 點 Scan → 選 repo 根資料夾（含全部分類那層）→ Godot 遞迴掃描所有 project.godot，把全部 demo 列入清單。

方式 C

瀏覽器試玩

多數 demo 已匯出至 GitHub Pages，可直接在瀏覽器體驗（效能低於原生）。連結見 學習路徑頁 底部外部連結。

主場景判定：每個 demo 的進入場景定義在其 project.godot 的 [application] → run/main_scene。除錯時先讀 project.godot 確認主場景與 Autoload，再看主場景掛載的腳本。
