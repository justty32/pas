核心亮點（5 大特色）
Plugin 抽象

Plugin Wrapper 層 + 約定式自動註冊

每個第三方 plugin 都再包一層 Wrapper，用 enum + 預載 Resource 取代易錯的裸字串 API。 設定系統的 ConfigurationControllerLoader 更靠「節點命名約定」自動把 enum 映射到節點，新增設定免改 Loader。

全域通訊

SignalBus 觀察者解耦

SignalBus 只宣告 signal、不含邏輯。守則：父子用一般 signal，跨層級 / 全域才走 SignalBus， 避免到處 get_node() 直連造成耦合。

UI 架構

Component-Driven UI（Builder 注入）

UiBuilder 遞迴掃描可聚焦的 Control，自動注入 TwistMotion（hover 果汁感動畫）與 ControlFocusOnHover（控制器焦點），模擬 Godot 未來的 Traits。放任何按鈕進選單即享一致 UX。

存檔

反射式存檔 + §§§ 簽章

SaveData 以 get_script_property_list() 掃描非底線變數自動序列化，新增欄位免寫存取碼。 落地為 JSON 並附 §§§ 簽章，偵測 OS 寫入殘留的損毀尾段；支援自動存檔、加密、base64 匯入匯出。

Web 整合

HACKS / Web 剪貼簿 JS 注入

把 Godot 已知 issue 的 workaround 文件化。重點是用 JavaScriptBridge 注入原生 HTML/CSS/JS 對話框， 繞過 web 無法存取剪貼簿（#81252），讓存檔字串可複製分享，並把遊戲 Theme 顏色橋接進 CSS。

雙持久化

設定 / 存檔分流

應用設定走 INI（ConfigStorage → user://config.cfg），遊戲存檔走 JSON（多槽 + 簽章 + 可選加密）。兩套獨立機制，皆以「約定優於設定 + 反射/命名自動化」壓低擴充成本。

場景流（啟動 → 選單 → 遊戲）
BootSplashScene（主場景） → MenuScene（visible 切子選單） → SaveFilesMenu 選檔 → GameScene（動態載入 GameContent） → PauseMenu / Leave

BootSplash _ready() 即以 fade_boot 轉場到主選單；GameScene 依 GameOptions 的 Game Mode 動態實例化玩法（2D clicker / 3D FP / 空），離開時先存檔再轉回選單。詳見架構頁 Level 3。
