# 基礎知識：四足機器人運動學 (Locomotion 101)

四足機器人的運動比輪式機器人複雜得多，主要涉及以下三個領域：

## 1. 逆向運動學 (Inverse Kinematics, IK)
*   **正向運動學 (FK)**：已知三個舵機的角度，求腳尖在哪裡（簡單）。
*   **逆向運動學 (IK)**：已知腳尖想去哪裡，求三個舵機該轉幾度（困難，但這是走路必須的）。
*   **DoF (自由度)**：本專案每條腿有 3 個舵機，代表有 3 個自由度，足以讓腳尖到達空間中任何一點。

## 2. 步態 (Gait)
步態是四條腿移動的時序邏輯。
*   **支撐相 (Stance Phase)**：腳踩在地上，支撐身體並向後推。
*   **擺動相 (Swing Phase)**：腳抬離地面，向前跨步。
*   **Trot (對角步)**：左前/右後腿一組，右前/左後腿一組，交替移動。這是小型機器狗最穩定的快速步態。
*   **Walk (常步)**：一次只抬起一條腿，隨時保持三條腿在地，最安全但最慢。

## 3. 平衡 (Balance)
*   **靜態平衡**：重心 (CoM) 的垂直投影必須落在支撐腿構成的多邊形內，機器人才不會倒。
*   **動態平衡**：考慮到運動產生的慣性。就像騎腳踏車，雖然靜止會倒，但動起來就能保持平衡。本專案目前主要實現的是初級的靜態平衡。
