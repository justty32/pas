# 基礎知識：機器人學中的數學與幾何

開發機器狗不需要成為數學家，但必須掌握以下三個核心概念，它們是逆向運動學 (IK) 的基石。

## 1. 三角函數 (Trigonometry)
在機器狗的腿部運動中，我們不斷在處理「角度」與「長度」的轉換。
*   **Sin / Cos / Tan**：已知角度和斜邊長度，求直角三角形的對邊或鄰邊。
*   **Asin / Acos / Atan2**：已知邊長，反過來求角度（這就是 IK 的核心）。
    *   *注意*：`atan2(y, x)` 比 `atan(y/x)` 更強大，因為它能處理 $x=0$ 的情況並正確判斷正負象限。

## 2. 餘弦定理 (Law of Cosines)
這是計算大腿和小腿角度的最重要公式。當你知道三角形的三條邊長（大腿長、小腿長、以及髖部到足端的距離），就能算出夾角：
$$c^2 = a^2 + b^2 - 2ab \cos(C)$$
在程式碼中，它被用來求出 $\beta$ 和 $\gamma$ 角。

## 3. 座標系 (Coordinate Systems)
*   **全域座標系 (World Frame)**：以大地或機器人初始位置為基準。
*   **本體座標系 (Body Frame)**：以機器狗的中心為基準。
*   **腿部局部座標系 (Leg Local Frame)**：以每條腿的肩膀（髖關節）為基準。
*   **座標轉換**：機器狗移動時，我們需要透過「旋轉矩陣」將目標點從全域座標轉換到腿部局部座標，這樣 IK 演算法才好算。
