# 基礎知識：電子電路與舵機驅動 (Servos & Power)

硬體是軟體的載體，電路設計不當會導致軟體再好也動不起來。

## 1. 舵機 (Servo Motor) 的運作
*   **內部結構**：直流馬達 + 減速齒輪組 + 電位器（回饋角度）+ 控制電路。
*   **MG90D 的局限**：這類廉價舵機的內部電位器解析度有限且存在非線性。本專案的「分段校準」就是為了解決這個問題。
*   **死區 (Dead Band)**：如果目標角度變化太小，舵機可能不會動，這是為了保護馬達不被頻繁微調燒毀。

## 2. 電源管理 (Power Management)
控制 12 顆舵機對電源是極大的挑戰：
*   **瞬間電流 (Inrush Current)**：當機器狗跳躍或啟動時，12 顆舵機可能瞬間抽走 5A 以上的電流。
*   **壓降 (Voltage Drop)**：大電流會導致電池電壓暫時下降。如果降得太厲害，ESP32 就會重啟（這就是為什麼很多人覺得代碼有問題，其實是電路撐不住）。
*   **隔離設計**：建議將 ESP32 的電源與舵機的電源分開（或加強電容濾波），防止馬達產生的電磁雜訊干擾控制器。

## 3. 18650 鋰電池
*   **電壓範圍**：充滿 4.2V，放空 3.0V。兩節串聯（2S）提供約 7.4V，這對大多數舵機來說是合適的，但需要穩壓器降壓給 ESP32 (3.3V) 使用。
