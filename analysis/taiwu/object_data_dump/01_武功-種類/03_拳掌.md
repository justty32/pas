# 太吾繪卷 · 武功種類 — 拳掌（88 個）

> 來源：名稱取自安裝版 `StreamingAssets/ConfigRefNameMapping/*.ref.txt`（與遊戲同步），品級／數值取自反編譯 `Assembly-CSharp/Config/*.cs` 的硬編資料，以 TemplateId 對接。抽取工具：`extract_objects.py`。

> 分類鍵＝`CombatSkillItem.Type`(14 類)。**正特效**＝順練效果、**逆特效**＝逆練效果（來自 `SpecialEffect` 表，由武功的 DirectEffectID/ReverseEffectID 連結）；**典故**為功法文學介紹(Desc)。特效文字中的 `$0$`/`$1$` 是遊戲即時計算的「威力成數」佔位，靜態資料無法還原，保留原樣。

## 品級 2（10 個）

### 大通臂拳（id 323）
- **正特效**：以此功法攻击敌人时，敌人的破体越高，敌人的御体就减少越多
- **逆特效**：当施展此功法需要的式不足时，运用者可将其作为腿法施展，能够影响腿法的效果此时均会对此功法产生影响；此功法发挥$0$成威力后，运用者获得2层此效果：运用者每累积后退1距离，就会从施展进度50%开始，无消耗施展此功法攻击敌人，以此效果施展的此功法不会获得摧破效果层数
- 典故：“其起如虎扑人，其落如鹰捉物”，力贯双臂，左右一体，谓之“通臂”也。大通臂拳招式刚猛，直上直下，放长而击远，势整而力顺。

### 十二手滥缠丝拳（id 331）

- 典故：十二手滥缠丝拳行拳如春蚕吐丝，发放如毒箭离弦。诀曰：“脚踏长山手拉弓，巴拿抖造力无穷，擒拿挂劈缠丝劲，直踏中宫冲洪门”。

- 典故：六十八式罗汉拳模仿十八罗汉之姿态神通，既有灵活多变之招式，又不离三正守中之根本，可谓刚柔并济，神形完备。
