# 太吾繪卷 · 武功種類 — 內功（168 個）

> 來源：名稱取自安裝版 `StreamingAssets/ConfigRefNameMapping/*.ref.txt`（與遊戲同步），品級／數值取自反編譯 `Assembly-CSharp/Config/*.cs` 的硬編資料，以 TemplateId 對接。抽取工具：`extract_objects.py`。

- **逆特效**：此天赋无法被封禁；蒲牢无法同时受到一种以上类型的直接伤害；蒲牢受到直接外伤后，在受失神、直接内伤前，免受任何直接外伤伤害；蒲牢受到直接内伤后，在受失神、直接外伤前，免受任何直接内伤伤害；蒲牢受到失神伤害后，在受直接外伤、内伤前，免受任何失神伤害

### 十五古法（id 915）
- **正特效**：为敌种下王蛊
- **逆特效**：为敌种下王蛊
- 典故：复生之人所习得的武学种类繁多，驳杂混乱，多为强行灌注于体内而成。此人虽曾功力尽失，但如今受死气、魔气的驱使，那些玄妙的法门，便又生出了诡秘的功力。在死气、魔气的驱使下，这些功法虽然威力奇大，且能够不拘常理的运用，但在遇到真传正法之时，便会显露出致命的破绽。
