核心價值 — Lisp 巨集元編程
Tutorial 04

用 Lisp 巨集做元編程（C-Mera 真正的價值）

這是 C-Mera 值得使用的核心理由。defmacro 在展開期執行任意 Lisp 程式以產生 C 程式碼——比 #define 強到不是同一個維度。涵蓋 hygienic macro、lisp 逃生艙、cintern/symbol-append 識別字操作、自動 dispatch table、程式碼生成迴圈。

defmacro hygienic macro (lisp ...) 逃生艙 cintern / symbol-append dispatch table 自動生成 展開期迴圈
→ 04_macros_the_fun_part.md

為什麼巨集這麼重要？

C 的 #define 是字串替換，有重複計算、無衛生性等缺陷。C-Mera 的 defmacro 在展開期執行完整的 Common Lisp 程式：你可以迴圈、查表、操作符號、產生任意 sexp。這讓「能用 C 的效能，卻有 Python metaclass 的彈性」成為可能。
