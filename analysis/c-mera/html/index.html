C-Mera

以 Common Lisp 驅動的 S-運算式 → C 家族語言 源到源編譯器
讓 Lisp 巨集成為 C/C++/CUDA/GLSL/OpenCL 的元編程引擎

— analysis/c-mera · 架構分析 + 教學文件 —

Common Lisp source-to-source C / C++ CUDA / OpenCL GLSL SBCL · CCL · ECL
C-Mera = 一個讓你用 Lisp S-運算式撰寫 C/C++ 程式的工具；它不引入抽象層，sexp 與目標語言語意一對一對應，但你能用完整的 Common Lisp 巨集做 C 語言做不到的元編程。
編譯流水線
.lisp 輸入 → Reader（預處理） → 巨集展開 → AST 節點 → Traverser 重寫 → Pretty Printer → .c / .cpp / .cu 輸出
支援語言後端

C

cm c input.lisp -o out.c
基礎後端，其他後端均繼承自此。

C++

cm c++ input.lisp -o out.cpp
繼承 C 後端，加入 class、template、lambda、exception、cast 家族。

CUDA

cm cuda input.lisp -o out.cu
繼承 C++，加入 launch、threads/blocks、共享記憶體。

GLSL / OpenCL

cm glsl / cm opencl
繼承 C，加入 shader layout / OpenCL 特定語法。
