技能、插件與自註冊體系

解構 hermes-agent 的模組化設計，如何透過 Skills 與 Plugins 實現功能無限擴充。

擴充雙軌制

技能系統 (Skills)

基於腳本的輕量化功能。支援「自我維護」，Agent 可以自主產出、修改並 Patch 這些技能。透過 SKILL.md 定義 Metadata 與依賴。

插件系統 (Plugins)

系統級的重型擴充。可以透過 Entry-points 載入，具備完整的 VALID_HOOKS 生命週期鉤子，能介入 API 請求、LLM 輸出與閘道分派。

技術細節：插件安全與載入

懶加載單例 (Lazy Singleton)

plugins/plugin_utils.py 提供了 lazy_singleton 裝飾器。它解決了插件開發中最常見的「併發初始化」競態條件問題，確保昂貴資源（如 DB 連線）在多執行緒環境下僅被建立一次。

插件鉤子清單 (Hooks Matrix)

鉤子名稱 觸發時機 應用場景
transform_llm_output 模型生成文字後，輸出前 即時翻譯、過濾敏感詞、角色扮演強化。
pre_tool_call 執行工具前 安全審計、參數自動修正。
pre_gateway_dispatch 外部訊息進入閘道時 訊息路由、多平台認證對接。
