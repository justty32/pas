# 進階據點：權威與領導力系統 (Authority & Leadership System)

在「模擬市民模式」下，玩家對盟友 NPC 的指揮權並非恆定，而是與角色在世界中的社會地位、軍事階級及個人聲望深度掛鉤。這創造了一種從「觀察者」到「統治者」的成長曲線。

## 1. 權威等級 (Authority Tiers)

玩家角色在特定據點或派系中的地位決定了其基本指揮權限：

*   **一級：外來訪客 (Guest / Stranger)**
    *   **控制權**: 無。
    *   **交互**: 只能進行交易與接受任務。
*   **二級：受任冒險者 (Trusted Ally / Contractor)**
    *   **控制權**: 僅限任務目標。
    *   **交互**: 可以請求 (Request) 特定的 NPC 協助完成某個任務小隊。
*   **三級：受勳軍官 (Officer / Knight)**
    *   **控制權**: 戰鬥徵調權。
    *   **交互**: 在襲擊發生時，可以「徵調 (Draft)」一定數量的民兵或低階士兵進入手動操作模式。
*   **四級：據點領袖 (Governor / Lord)**
    *   **控制權**: 完全掌控權（可切換）。
    *   **交互**: 可以在「自主生活」與「手動管理」模式間自由切換。NPC 會絕對服從命令。

## 2. 聲望與指揮效力 (Reputation & Command Efficacy)

即使具備職位，NPC 的執行效率仍受聲望影響：

*   **個人名望 (Renown)**: 透過完成公會任務獲取。名望越高，NPC 在非戰鬥狀態下越傾向於主動協助玩家。
*   **領袖魅力 (Charisma)**: 玩家角色的社交屬性與特質。高魅力的角色在發出徵調指令時，NPC 的心情懲罰較低，且不容易發生「違抗命令」的崩潰情況。

## 3. 緊急徵調機制 (Emergency Draft Logic)

在極端危機（如基地即將被攻破）時，系統提供動態的指揮權擴張：

*   **一呼百應 (The Hero's Call)**: 若玩家角色聲望極高且戰況慘烈，可啟動一次性的「全體徵調」。此時所有自主 NPC 會暫時受玩家控制。
*   **代價與反彈**: 越權指揮或強制讓不擅戰鬥的 NPC 衝鋒會導致嚴重的聲望損失，甚至引發據點內的「政治彈劾」或「暴動」。

## 4. 指揮權的視覺反饋 (Visual Feedback)

*   **光環效應**: 被徵調的 NPC 腳下會出現特定顏色的光環，代表其當前處於玩家的「指揮鏈」中。
*   **指令 UI**: 根據地位高低，右鍵選單的指令強度會從「請求幫忙」演變為「強令執行」。

## 5. 總結：權力與責任的平衡

這套系統讓「指揮權」本身成為了一種遊戲資源與成長目標。玩家必須透過在據點生活、完成任務、建立功勳，才能逐步換取對這群自主盟友的控制力。這讓 RimWorld 的後期玩法從單純的物資累積，轉向了對「影響力」的經營。
