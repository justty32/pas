插件、有線鉤子與 MCP

解構 OpenClaw 的模組化體系，如何實現跨平台的無縫擴充。

插件與有線鉤子 (Wired Hooks)

Wired Hooks

明確的生命週期鉤子，允許插件在訊息認領、LLM 調用、回覆分派等關鍵節點進行精準介入。

MCP 整合

原生支援 Model Context Protocol，讓 Agent 能直接呼叫外部工具伺服器提供的功能。

關鍵源碼路徑

  • src/plugins/registry.ts - 插件註冊與載入
  • src/plugins/wired-hooks-*.ts - 各類生命週期鉤子
  • src/mcp/ - MCP 協議實作
