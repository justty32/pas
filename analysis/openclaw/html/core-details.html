AI 記憶：QMD 與耐久文件

QMD 向量引擎

基於 @tobilu/qmd 的本地向量檢索。支援會話轉錄的自動索引與語義搜尋 (vsearch)，讓 Agent 具備「回憶」能力。

MEMORY.md

工作空間的「真相之源」。Agent 啟動時強制載入的耐久記憶文件，確保行為與偏好的一致性。

AI 決策：思考分級制度

透過 Thinking Profiles 實現推理能力的可量化調度：

  • 五級推理：從 off (直覺反應) 到 xhigh (深度推理)。
  • 模型適配：動態映射各供應商的原生推理參數。

安全網：溯源與隔離

輸入溯源 (Provenance)

識別訊息來源並注入安全前綴。徹底阻斷跨會話的指令劫持風險，確保 AI 不會被誤導執行來自非人類的惡意指令。

硬化沙盒 (Sandboxing)

支援 Docker/SSH 隔離。實作了路徑錨定保護與工具執行審核流程 (Exec-Approval)，防止 Agent 逃逸並損毀主機系統。
