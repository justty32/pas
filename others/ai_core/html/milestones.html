骨架與協議

預估：半天～一天 待開始

  • ⬜6 個 CLI entrypoint 空殼（ai-core-server/call/hub/hub-server/sfc/author）

M3.5

Agent Docs 自動化

待開始
  • ⬜ai-core-hub --gen-agent-md > AGENTS.md（通用 agent 入口文件）
  • ⬜ai-core-hub --gen-functions-md > auto/FUNCTIONS.md（函式清單）
  • ⬜ai-core-hub --export claude-skill → auto/skills/<name>/SKILL.md
  • ⬜author 成功 register 後自動觸發 gen-functions-md + gen-agent-md
  • ⬜auto/ 納入 CI / pre-commit 驗證（與 funcs/ 一致性）
📋 auto/ 目錄讓 AI agent 能隨時取得最新函式清單，是 Heuristic Learning 飛輪的重要基礎設施。

待開始
