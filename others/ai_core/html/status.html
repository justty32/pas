tools/ — 工具端（7 個工具）

indexer.py
掃描資料夾的可執行檔，呼叫 --metadata，彙整成靜態索引（JSON/markdown）
router.py
讀 JSON 設定檔，name → 可執行物 mapping，subprocess dispatch（透傳 stdin/stdout/exit）
switch.py
有條件邏輯的 router；依「switch 變數值」走規則表分支到不同 target
sfc.py
SFC 完整實作 Layer 0–4；git-style subcommand CLI；已改用 meta_core 解 Gap A/B
meta_core.py
Gap A/B 修法提案原型：放寬版 --metadata 攔截 + subcommand-scoped metadata
hub.py
函式生態 → 給 LLM 的 skill 清單；context budget 逐級收斂；scan/budget/format
llm_entry_manager.py
LLM Entry Manager 原型：singleton server + consume rate + mock backend
chain.py
宣告式 pipeline（JSON spec）；--derive 從各 stage --metadata 推導複合 metadata

lib/ — 可重用 library（11 個模組）

state_dirs.py
複合規範「標準狀態目錄慣例」：解析 .config/.cache/.state/.data 路徑、declared() 產 register 片段
recovery.py
複合規範「中斷恢復慣例」：recovery.json manifest、resume/rollback/reset、session() context manager
memoize.py
記憶化快取：sha256 key 組成、.cache/<name>/<hash> 儲存、失效策略（D1 已拍板）
server.py
persistent server 標準 lifecycle（NDJSON）；forge 的迴圈泛化版
singleton.py
singleton 資源：RateMeter（多維 consume）+ RequestQueue（enqueue/dequeue/cancel）+ SingletonResource
trace.py
調用鏈：結構化 stderr log + trace id 經 env 傳遞 + Collector 重建調用樹
call.py
跨邊界統一呼叫：InProcess / Subprocess / Http 同一個 call(text)→text 介面
llm_call.py
llm_call(str)→str 基底 + bind() context packing；可插拔 backend（預設 mock）
compose.py
多函數合作組合子：pipe / fanout / route / decompose + retry_until_valid / vote / best_of / guard
interact.py
多函數交互：blackboard driver + actor_critic（LLM-as-judge）+ debate；max_rounds 安全閥
compose_meta.py
組合的軸推導規則：MetaFn + mpipe/mfanout_reduce 自動算複合 metadata（A4 懸案）

demos/ — 端到端 Demo（3 個）
