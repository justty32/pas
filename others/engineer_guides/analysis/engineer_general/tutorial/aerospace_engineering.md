# 航太工程 (Aerospace Engineering) 入門指南

## 領域簡介
航太工程是關於飛行載具（飛機、無人機、火箭、衛星、太空船）的研究與開發。它是機械工程的進階延伸，專注於極端環境（高空、真空、高熱）下的載具效能、結構穩定性與自動導航。

## 課綱與知識列表

### 1. 高中階段 (基礎與準備)
*   **物理基礎：** 牛頓運動定律、流體概念。
*   **數學基礎：** 空間向量、三角函數、微積分初步。
*   **基礎素養：** 英文讀寫能力（航太領域全球標準語言）。

### 2. 大學階段 (核心工具與理論)
*   **四大支柱：**
    *   **空氣動力學：** 流場分析、升力與阻力原理。
    *   **航太結構與材料：** 輕量化結構、材料力學。
    *   **推進系統：** 噴射引擎原理、火箭推进。
    *   **飛行力學與控制：** 飛行穩定性、自動導航控制。
*   **整合技能：** 飛機/火箭初步設計、模擬軟體 (MATLAB, CATIA, ANSYS)。

### 3. 碩士階段 (專業深耕與應用)
*   **研究分組：**
    *   **熱流與推進：** 高等 CFD、燃燒動力學、高超音速流。
    *   **導航控制：** 衛星姿態控制、無人機集群演算法。
    *   **太空工程：** 軌道力學、太空環境影響分析。
*   **實務實習：** 飛機維修實務、航太零件製造技術。

### 4. 博士階段 (原創研究與創新)
*   **尖端研發：** 再生火箭技術、深太空探測動力系統、隱形技術。
*   **理論突破：** 極超音速流體模型、非線性導航控制理論。
*   **戰略領導：** 領導國家級航太計畫，推動太空探索邊界。
